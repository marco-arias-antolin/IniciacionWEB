Empezamos con los estilos:

Se crea una hoja de estilos en la carpeta css: css/styleEjemplo2.css

Se enlaza el archivo html con la hoja de estilos css.

<link rel="stylesheet" href="css/styleEjemplo2.css">

En VSCode se puede hacer primero el link y al hacer ctrl+click encima de "css/styleEjemplo2.css" te crea el archivo.

dentro del archivo metemos el siguiente código css

h1 {
  font-family: Arial, Helvetica, sans-serif;
  /* Posibles unidades: px, em, etc */
  font-size: 24px;
  color: #a712e2;
}
  • font-family: define la fuente (tipo de letra).
  • font-size: define el tamaño de la fuente.
  • color: define el color de la fuente.

Se puede asignar un estilo a una clase poniendo .nombreClase

HTML

<p class="miEstilo1">Párrafo con la clase miEstilo1</p>

CSS

.miEstilo {
  color: purple;
  text-align: start;
  width: 100%;
}

Párrafo con la clase miEstilo1

Para poner un texto en negrita se encierra en la etiqueta

<span class="negrita"></span>

Y creamos la clase negrita en el CSS

.negrita {
  font-weight: bold;
}

Ejemplo en un párrafo:

Carles Duran tiene claro que “es muy importante ganar los partidos de casa, ya se está viendo que marcan la diferencia. Nosotros lo valoramos mucho, porque el equipo siempre después de una derrota necesita el apoyo de la gente. Y tengo claro que el equipo debe hacer para que la gente recupere la confianza que en el último partido no les dimos. Lo que queremos es volver a jugar bien a baloncesto”, dijo el técnico.

La cursiva y el subrayado se usan como la negrita (usando la etiqueta "span") y el css sería:

.cursiva {
  font-style: italic;
}

.subrayado {
  text-decoration: underline;
}

Ejemplo de cursiva y de subrayado

Se pueden aplicar varias clases en una etiqueta.

<p class="negrita cursiva subrayado">Varias clases en una etiqueta.</p>

Para cambiar el color de fondo se usa:

background: black;
